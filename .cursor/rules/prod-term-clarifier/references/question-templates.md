# 質問テンプレート

## 1. 追加 vs 置換

```text
確認したい点:
「[機能名]を作る」はどちらですか？
A. 既存機能を残して新規追加
B. 既存機能を置き換え
```

## 2. 非表示 vs 削除

```text
確認したい点:
「不要」の意味はどちらですか？
A. UI上で非表示（コードは維持）
B. コード/資産を削除
```

## 3. 移行方式

```text
確認したい点:
移行方式はどちらですか？
A. 共存期間あり（段階移行）
B. 即時切替（旧機能停止）
```

## 4. 同等の範囲

```text
確認したい点:
「同等」はどの範囲ですか？
A. UIのみ同等
B. 機能まで同等
C. データ互換まで同等
```

## 5. 今回ケース用（アンケート/AI診断）

```text
確認したい点:
1) 「アンケート機能を作る」はどちらですか？
   A. AI診断テストを残したまま、アンケートを追加
   B. AI診断テストを削除して、アンケートへ置換
2) ナビ表示はどちらですか？
   A. TESTとSURVEYを両方表示
   B. TESTのみ表示（SURVEYはURL配布のみ）
```

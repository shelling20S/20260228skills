## 1. マインドセット

- **批判禁止**: 「それは難しい」「コストが...」は後で。今は量を出す
- **便乗歓迎**: 他の案から派生してOK
- **突飛さ歓迎**: 現実的かは後で考える
- **量が質を生む**: 10案より20案、20案より50案

## 2. 発散テクニック

状況に応じて使い分ける。

### 軸をずらす
```
「誰に？」を変える → 学生向け、シニア向け、B2B向け...
「いつ？」を変える → 朝専用、週末専用、年1回...
「どこで？」を変える → 自宅、移動中、海外...
「なぜ？」を変える → 節約、自己実現、承認欲求...
```

### 逆転思考
```
「最悪のサービスは？」→ 逆にすると？
「絶対やってはいけないことは？」→ あえてやると？
「競合がやらないことは？」→ なぜやらない？やる価値は？
```

### アナロジー
```
「〇〇業界の△△を、この領域に持ち込むと？」
「Uber for X」「Airbnb for X」パターン
異業種の成功パターンを借りる
```

### 制約追加
```
「スマホなしで実現するなら？」
「1人で運営するなら？」
「無料で提供するなら？」
```

### 10x思考
```
「10倍速くするには？」
「10分の1のコストにするには？」
「10倍の顧客に届けるには？」
```

## 3. 出力フォーマット

### 案の束

```markdown
## アイデア一覧

### カテゴリA: [カテゴリ名]
1. **[案名]**: [1行説明]
2. **[案名]**: [1行説明]
3. ...

### カテゴリB: [カテゴリ名]
4. **[案名]**: [1行説明]
5. ...

---

## 次に深掘る3つ

以下を選定理由とともに提案:

1. **[案名]** - [選定理由: 新規性/実現性/インパクトなど]
2. **[案名]** - [選定理由]
3. **[案名]** - [選定理由]

深掘りたいものを選んでください。または「もっと案を出して」「別の切り口で」も可。
```

## 4. 会話例

**ユーザー**: 飲食店向けのSaaSでアイデア出ししたい

**Claude（発散モード）**:
```
発散モードで進めます。まず思いつくままに案を出していきますね。

## アイデア一覧

### カテゴリA: 業務効率化
1. **AI発注アシスタント**: 売上予測から自動発注提案
2. **シフト最適化**: 混雑予測×スタッフスキルでシフト自動生成
3. **レシピ原価管理**: 仕入れ値変動を自動反映

### カテゴリB: 集客・マーケ
4. **空席リアルタイム広告**: 空席時に周辺ユーザーへプッシュ
5. **常連スコアリング**: 来店頻度×単価でVIP自動認定
6. **レビュー返信AI**: Google/食べログの返信を自動生成

### カテゴリC: 顧客体験
7. **待ち時間エンタメ**: 待ち時間にクーポン付きミニゲーム
8. **アレルギー自動チェック**: 注文時にアレルギー成分警告
9. **おまかせAI**: 好みを学習して本日のおすすめ提案

[... 続く ...]

---

## 次に深掘る3つ

1. **AI発注アシスタント** - 食品ロス削減という社会課題に直結
2. **空席リアルタイム広告** - 即効性あり、ROI測定しやすい
3. **常連スコアリング** - CRM基盤として横展開可能

どれを深掘りますか？または「もっと集客系で」「小規模店向けに絞って」など方向性を指定してください。
```
